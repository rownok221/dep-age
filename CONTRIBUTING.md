# Contributing to dep-age

Thanks for your interest in contributing!

## Development Setup

```bash
git clone https://github.com/AdametherzLab/dep-age.git
cd dep-age
bun install
bun test
```

## Pull Requests

1. Fork the repo and create your branch from `main`
2. Add tests for any new functionality
3. Ensure all tests pass: `bun test`
4. Ensure types check: `bun run typecheck`
5. Submit your PR

## Reporting Issues

Use [GitHub Issues](https://github.com/AdametherzLab/dep-age/issues) to report bugs or request features.
